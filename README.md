# xbash
**bash eEnhanced tools**

### Compile the script and generate binary files
```bash
xbash -X example.sh
```
The make code is ```example.sh.x```，Add after the original file name```.x```。

### Execute binary files

xbash example.sh.x

You can also add shebang to the .x file, such as ```#!./xbash```, ```chmod +x``` example.sh.x are executed as separate programs and behave similarly to regular shells.

```xbash``` only enhances the functionality of the original ```bas```h and can be used as a regular ```bash```.

```xbash_android_v7a```  Android version, add execution permission after downloading

```xbash_linux_x86-64``` LINUX 64 bit version
