# xbash
**bash enhanced tools**

### Compile the script and generate binary files
```bash
xbash -X example.sh
```
The make code is ```example.sh.x```, Add after the original file name```.x```。

### Execute binary files
```bash
xbash example.sh.x
```
You can add shebang in the first line of the .x file, such as ```#!./xbash```, ```chmod +x example.sh.x``` as executable programs behave similarly to regular shells.

```xbash``` only enhances the functionality of the original ```bash``` and can be used as a regular ```bash```.

```xbash``` is non open source software, and executable permissions need to be added after downloading

```xbash_android_v7a```  Android version, add execution permission after downloading

```xbash_linux_x86-64``` LINUX 64 bit version
