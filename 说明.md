# xbash
**bash功能增强工具**

### 编译脚本并生成可二进制文件
```bash
xbash -X example.sh
```
生成的代码是 ```example.sh.x```，即原始文件名后加```.x```。

### 执行二进制文件

xbash example.sh.x

可以在.x文件首行中加shebang, 如#!./xbash, chmod +x example.sh.x 做为单独程序执行,与普通的shell行为类似。

xbash只对原bash进行了功能增强，可当做普通的bash使用。

```xbash_android_v7a``` 安卓版，下载后加可执行权限

```xbash_linux_x86-6```4 LINUX 64位版本
