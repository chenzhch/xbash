# xbash
**bash语言转换为C代码的工具**

### 安装

可直接使用现有可执行程序，或通过源码编译安装，步骤如下：
```bash
tar xvf bash-5.2.37.tar.gz
cp command.h execute_cmd.c shell.c  ./bash-5.2.37 
cd bash-5.2.37
./configure
make
mv bash xbash
```

### 使用

```bash
chmod +x xbash
tar xvf bash-5.2.37.tar.gz
cd bash-5.2.37
./configure LDFLAGS="-static"
修改make.sh文件中BASHDIR和XBASH,指向实际值
```

### 编译脚本并生成可二进制文件
```bash
sh make.sh example.sh
```
建议用```-static```静态编译和```strip```去掉符号表，增加逆向难度。对编译后的程序进行全面测试，以防止生产故障。  


### 执行二进制文件
```bash
./example.sh.x
```
### 特性

- 转为C代码，执行时不会转回原脚本代码

- 生成独立可执行程序，不依赖于```bash```解释器 

- 解决[decrypt](https://github.com/chenzhch/decrypt)所有破解方法
